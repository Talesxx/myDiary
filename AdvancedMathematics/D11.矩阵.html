矩阵的加法和减法 ​

1.1 矩阵的加法 ​

对应元素相加。 例如： [111111]+[023−111]=[134022]

显然，只有同型矩阵才能相加。

1.2 矩阵的减法 ​

对应元素相减。 例如： [123333444]−[101001000]=[022332444]

同理，也只有同型矩阵才能做减法。

1.3 五条运算律 ​

① A + B = B + A ② ( A + B ) + C = A + ( B + C ) ③ A + O = A ④ A + ( − A ) = O ⑤ A + B = C    ⟺    A = C − B

上述矩阵A 、 B 、 C 和 O A、B、C和OA、B、C和O在运算时必须为同型矩阵。

2 矩阵的数乘 ​

2.1 数乘的定义 ​

用一个数乘以矩阵等于这个数乘以矩阵的所有元素。 例如： k[123456789]=[1k2k3k4k5k6k7k8k9k]

2.2 三条运算法则 ​

① k ( A + B ) = k A + k B ② ( k + l ) A = k A + l A ③ k l ( A ) = ( k l ) A
